Quick Start: 7 Steps to Your First Campaign

1) Prepare the foundation

Create a Business Manager, connect your Page and Pixel, verify domain, and set up payment.

2) Define your objective

Choose a goal that matches your outcome: Awareness, Traffic, Leads, Sales, or Engagement.

3) Choose your audience

Start broad with location, age, and interests. Add exclusions. Use Custom and Lookalike Audiences as data grows.

4) Pick placements

Begin with Advantage+ placements. Later, analyze performance by placement and refine if needed.

5) Set budget and schedule

Use daily budget sized for 50+ target conversions/week (if possible). Run continuously at first.

6) Create compelling ads

Use 1–3 clear value propositions, strong visuals, social proof, and a direct call-to-action.

7) Launch and monitor

Let the learning phase run (3–5 days). Optimize only after you have statistically meaningful data.

Creative Best Practices

  • Lead with outcome: show the “after state” your product creates.
  • Use mobile-first formats: 1:1 or 4:5 images/videos, bold text, large subtitles.
  • Test creative in batches: 3–5 variations per ad set (headline, image, hook).
  • Keep copy structured: Hook → Value → Proof → CTA. Avoid walls of text.
  • Refresh winners every 2–4 weeks to prevent ad fatigue.

Optimization Checklist

Before editing

Wait for 50+ conversions/ad set or 7 days. Avoid frequent changes during learning phase.

Scale winners

Increase budget 10–20% every 2–3 days, or duplicate to a higher budget ad set.

Fix underperformers

Check funnel drop-offs, landing speed, audience-message fit, and creative clarity.

Measure correctly

Use conversions API + pixel, set proper attribution window, and track blended ROAS.

Frequently Asked Questions

How much budget do I need?

Enough to achieve ~50 conversions per ad set per week. If not possible, start with what you can, focus on learning.

Which objective should I pick?

Match the conversion you care about (e.g., purchases → Sales). Don’t optimize for clicks if you want sales.

How many ads per ad set?

3–5 variations is a good starting point. Kill obvious losers after meaningful spend.

What metrics matter?

Cost per result, conversion rate, CTR, CPM, and blended ROAS. Trends > daily noise.

Tip: Keep your account healthy — pay on time, avoid disapproved policies repeatedly, and use verified domains.
